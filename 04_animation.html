04_ANIMATION

Introduction

For my "04_Animation" project, I created an interactive click-based game centered around the American election between Kamala Harris and Donald Trump. The game offers players a chance to explore and gain knowledge about the election, eventhough it is biased though engaging animations and decision-making elements.

My goal was to create an educational yet entertaining experience that reflects the importance of political figures. By combining dynamic visual effects with interactive gameplay, I was able to showcase my skills in animation, interactivity, and storytelling, making the project both fun and informative.

Front page

Front page from my 04_ANIMATION project.

Process

The development of "The Election" was a multifaceted process that involved several new skills. I began by learning JavaScript to build the interactive game logic, ensuring a dynamic user experience. To plan out animations and transitions, I used state machine diagrams, which helped me visualize the flow and anticipate user actions. CSS animations were then applied to create smooth, engaging effects throughout the game, adding to the overall interactivity.

Adobe Illustrator played a key role in designing custom graphics, while character design principles were used to develop visually clear and expressive characters which was then exported as SVG's. This enhanced the game’s storytelling and user interaction. I also created new game sounds in Adobe Audition, ensuring the sound design matched the atmosphere and theme of the game. This entire process not only improved my technical skills but also helped me understand how visual and auditory elements can enhance user engagement and the overall experience.

Product

The final product is a fun, interactive click-based game with a satirical twist on the American election between Kamala Harris and Donald Trump. Designed to be both engaging and educational, it lets players explore the two presidential candidates in the election through interactive animations. The simple yet clean graphics ensure an easy follow without distraction. The satirical tone brings humor to the experience, offering political insights.

The intuitive design makes it easy for players, regardless of knowledge, to understand and engage. The user interface enhances the experience with easy-to-read text, smooth transitions, and a layout that complements gameplay. Overall, the project merges humor, design, and interactivity for a memorable experience.
